RBNB DataTurbine

Web Browser Interface

Thank you for using DataTurbine, a software system that uses Ring Buffer Network Bus (RBNB) to connect data sinks to sources via buffered middleware. 

If you can see this message, you have installed the software correctly, and found the index page.  From here you can view the documentation, start applications and plugins, and view RBNB data.

Server

  • rbnbServer - Start an RBNB server on the webserver's machine.
  • WebCache - Start a WebCache on the webserver's machine.

Applications  (Note that these are started on the webserver's machine!)

  • rbnbAdmin - Start rbnbAdmin to view and configure an existing RBNB server.
  • rbnbPlayer - Start rbnbPlayer to control the playback of RBNB channels.
  • rbnbPlot - Start rbnbPlot to view RBNB channel data in real time.
  • rbnbSource - Start rbnbSource, the RBNB signal generator.
  • rbnbChat - Start rbnbChat, the RBNB chat room client.
  • RBNB messaging host - Start rbnbMsgHost, the RBNB messaging host.
  • FileDrop - Start FileDrop, the RBNB file drop application.
  • HttpMonitor - Start HttpMonitor, an application which copies files from any URL to any WebDAV server.

PlugIns

  • Robust Routing PlugIn - Routes data from remote server to local server. View running PlugIns and start new ones.
  • Virtual Source PlugIn - Map remote source.
  • Virtual Channels PlugIn - Map channels to another name.
  • Portable Network Graphic PlugIn - Converts time series data to a PNG format plot, viewable via most Web browsers.
  • Folder PlugIn - Post local disk files and directories as RBNB channels.
  • Caching PlugIn - Buffers remote data locally.
  • SQL PlugIn - Passes SQL statements to database, returns response.
  • ToString PlugIn - Converts data to ASCII string format.
  • ToHex PlugIn - Converts bytes to their hexadecimal string representation.
  • Proxy Source PlugIn - Simple streaming TCP/IP data source proxy.
  • Signature PlugIn - Sign and check signatures for authenticity.
  • Thumb Nail PlugIn - Convert a JPEG to a smaller thumbnail image.
  • Resample PlugIn - Change the sample rate of time series data.

Web Based Utilities

  • PlugIn Messenger - Sends a message to a PlugIn, such as rbnbChat or SQLPlugIn, and displays the response.

Documentation

  • Documentation Folder - Top level documentation folder.
  • RBNB Developer Guide - Documentation for RBNB developers, including an overview of RBNB clients and channel maps.
  • RBNB WebDAV Options Reference - Documents the request options used in conjunction with the RBNB WebDAV interface.
  • SAPI Javadoc - View the Simple API Reference Manual (javadoc) documentation.
  • ActiveX Interface - This manual explains the cross-language support offered on Windows platforms for the RBNB Simple API via the JavaBean to ActiveX bridge.
  • Servlet Deployment Guide - This is a guide for how to add your own RBNB servlets to the DataTurbine sytsem.
  • WebServer Docs - Documentation for the Jakarta-Tomcat servlet engine.

Online Viewers

  • RBNB WebDAV Interface - View RBNB channels via the WebDAV directory interface.
  • WebPlot: Web Based Data Viewer - Plot DataTurbine channels easily without additional software.
  • Portal to View and Search RBNB channels - This portal automatically generates URLs to view RBNB channels via a web browser.  These URLs may be pasted in web documents to create dynamic reports.
