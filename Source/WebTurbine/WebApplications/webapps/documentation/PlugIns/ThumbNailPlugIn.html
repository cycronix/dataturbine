Thumb Nail PlugIn Help

Thumb Nail PlugIn creates scaled versions (thumbnails) of JPEG images.

Configuration options for Thumb Nail PlugIn
RBNB Server Address The server to which the PlugIn connects.
PlugIn Name The registered name of this instance of ThumbNail PlugIn
JPEG Source/Channel An optional argument which specifies the Source/Channel of the JPEG images. If specified, only requests for scaled images from this Source/Channel will be served; the RBNB Server will not forward any other requests to the PlugIn.
Image Scale Factor                                         
A value from 0 - 4 which specifies the amount of scaling.  Values from 0 - 1 will produce an output image smaller than the input image; values from 1 - 4 will produce an output image larger than the input image.  A scale factor of 1.0 will keep the same image size.
Image Quality A value from 0 - 1 which specifies the quality of the output image.  A value of 0.0 corresponds to very high compression, very low quality.  A value of 1.0 corresponds to low compression, very high quality.
Maximum Number of Images For a multi-image request (that is, a PlugIn request with a non-zero duration) this parameter specifies the number of output images.  The timestamp of the images will be evenly spread over the duration.
