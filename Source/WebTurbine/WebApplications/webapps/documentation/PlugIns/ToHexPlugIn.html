Introduction

The ToHex plug-in converts raw bytes to their hexadecimal string representation. The display includes a header row, which numbers the hex values, and first column which contains the offset of the row of bytes from the start of the data. On the right is the ASCII representation of the bytes, if they represent printable characters.
